# Проверка покрытия

Автоматическая проверка должна подтвердить:

- каждый заявленный индикатор присутствует в `competencies.yaml`;
- каждый индикатор связан минимум с двумя различными оценочными средствами;
- каждая из 12 лабораторных работ связана минимум с одним индикатором;
- все лабораторные Markdown-файлы существуют.

Запуск: `python tests/`.
