# ADR-0002: оси и веса оценивания

- Статус: принято
- Дата: 2026-07-20

## Итоговая оценка дисциплины

| Средство | Вес |
|---|---:|
| Лабораторные работы | 20% |
| Тесты | 10% |
| Code-review и peer-review | 10% |
| Рубежная защита | 15% |
| Итоговый проект | 30% |
| Защита комиссии | 15% |
| **Итого** | **100%** |

## Итоговый проект

- программное решение — 70%;
- документация — 30%;
- нижний порог — 4 балла по каждой оси.

## Лабораторный блок

Взвешивание выполняется пропорционально нормативным часам РПД:
`ОС-1 = Σ(hᵢ·bᵢ)/Σhᵢ`, где `Σhᵢ = 28`.
