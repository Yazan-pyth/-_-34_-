# Информационные и материально-технические ресурсы

## Основная литература

1. Warden P., Situnayake D. *TinyML: Machine Learning with TensorFlow Lite on Arduino and Ultra-Low-Power Microcontrollers*. O'Reilly, 2020.
2. Banbury C. R. et al. “MicroNets: Neural Network Architectures for Deploying Tiny Machine Learning Applications on Commodity Microcontrollers.” *Proceedings of Machine Learning and Systems*, 2021.
3. Ren H. et al. “TinyML: A Comprehensive Survey of Machine Learning on Edge Devices.” *IEEE Internet of Things Journal*, 2023.

## Дополнительные материалы

- официальная документация RKNN-Toolkit2;
- ESP-IDF Programming Guide;
- FreeRTOS Kernel Reference Manual.

## Программное обеспечение

- TensorFlow, TensorFlow Lite, TensorFlow Lite Micro;
- PyTorch;
- ONNX и ONNX Runtime;
- PlatformIO и ESP-IDF;
- GCC for ARM;
- RKNN-Toolkit2;
- FreeRTOS;
- Armbian или Ubuntu для Rockchip;
- Docker;
- OpenCV;
- Edge Impulse;
- `perf`, `htop`, `tflite_model_analyzer` и средства профилирования RKNN.

## Аппаратные средства

- ESP32-S3;
- STM32;
- платформы Rockchip RK3588/RK3576, например Orange Pi 5;
- камеры, MEMS-микрофоны, IMU и акселерометры;
- осциллографы, логические анализаторы и датчики тока;
- GPU-сервер для предварительного обучения и сжатия моделей.
