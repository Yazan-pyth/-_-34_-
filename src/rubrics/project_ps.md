# Рубрика: программное решение

Оценка каждого критерия выставляется по ближайшему подтверждённому уровню. Промежуточные баллы 5, 7 и 9 используются, когда результат устойчиво находится между соседними портретами.

| Уровень | Описание |
|---:|---|
| 10 | Полностью воспроизводимое аппаратное решение с высокой практической ценностью или новизной; автоматизированы сборка, конвертация, деплой и профилирование; обоснован оптимальный компромисс качества, задержки, памяти и энергии. |
| 8 | Полный устойчивый прототип на целевой платформе; сравниваются варианты; измерения повторяемы; решения по оптимизации и надёжности обоснованы. |
| 6 | Самостоятельно реализован типовой end-to-end пайплайн; основные функции работают; есть базовые измерения и допустимое качество. |
| 4 | Минимальный сценарий выполняется по инструкции; модель запускается, но оптимизация, профилирование или обработка ошибок поверхностны. |
| 0-3 | Прототип не работает на целевом устройстве, результаты не проверяемы либо отсутствуют ключевые компоненты. |

## Интерпретация по критериям

- **PS1:** корректный инференс и базовый сценарий без внешней зависимости от ПК.
- **PS2:** корректная работа с GPIO/I2C/SPI/UART/DMA/сенсорами или NPU API.
- **PS3:** воспроизводимая конвертация, калибровка, квантование/прунинг и анализ потери качества.
- **PS4:** измерения на целевой платформе, методика измерения и сравнение baseline/optimized.
- **PS5:** watchdog, обработка сбоев, real-time ограничения, sleep-режимы или аргументированное исключение.
- **PS6:** требования и метрики связаны с выбранной отраслевой траекторией.
