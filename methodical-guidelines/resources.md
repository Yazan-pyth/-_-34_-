# Рекомендации преподавателям по использованию ресурсов

- Для трека TinyML/MCU используйте ESP32-S3 и, при наличии, STM32.
- Для трека Edge Linux/NPU используйте платформы Rockchip RK3588/RK3576.
- До работы на реальном оборудовании допускается предварительная отладка в QEMU или Wokwi.
- Версии библиотек, средств конвертации и драйверов фиксируются в отчёте.
- Для оценки производительности используйте измерения на целевом устройстве.
- Для лабораторных работ по энергопотреблению применяйте амперметр, датчик тока или осциллограф.
- Если физическое оборудование ограничено, распределяйте платформы между командами, но каждый обучающийся должен представить собственный отчёт и объяснить результаты.
